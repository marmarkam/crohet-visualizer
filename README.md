Crochet-Visualizer ultimately intends to be a program that can take input in the form of crochet instructions and output a visual representation of those instructions. This is specifically for patterns that work in the round, or for amigurumi.
While the currently available programs are sufficient for flat works, they do not suffice for amigurumi projects. These programs are typically restricted to color-work charts for crochet projects worked flat. There is no software (to my knowledge) that can represent works worked in the round. So, if a crocheter wants to make their own pattern for an amigurumi, they must go through a tedious trial-and-error process until they are happy with the final look. This project intends to be a tool of convenience for such crocheters.
